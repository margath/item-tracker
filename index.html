⚙️ Cloud Backend Setup

To save data across devices, you need a Google Apps Script to act as your database. Follow these steps:

  1. Create a new Google Sheet.
  2. In the top menu, click Extensions > Apps Script.
  3. Delete the default code and paste the script from the box below.
  4. Click Deploy > New deployment.
  5. Select type Web app. Set 'Execute as' to Me and 'Who has access' to Anyone.
  6. Click Deploy, Authorize the permissions, and copy the Web app URL.
  7. Paste that URL into the input field below!
